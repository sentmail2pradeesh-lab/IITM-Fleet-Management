# IITM Fleet Management System

## 📌 Overview
A web-based system to manage fleet vehicles, bookings, and users.

## 🛠 Tech Stack
- Frontend: (React / HTML / etc.)
- Backend: (Node.js / Flask / Django / etc.)
- Database: (MySQL / MongoDB / etc.)

## ⚙️ Features
- Vehicle booking
- Admin dashboard
- User authentication

## ▶️ How to Run

### Backend
cd Backend
npm install
npm start

### Frontend
cd Frontend
npm install
npm start
